Hoe werkt deze site?

Op deze pagina laten wij u stap voor stap zien hoe u een prachtig lichtgevend powerbankhoesje kunt maken. Hier kunt u een patroon maken, maar lees eerst deze website goed door.

Hoe werkt de site van micro:bit?

  1. U moet eerst een account aanmaken op de website van micro:bit. (Link naar micro:bit)
  2. Dan kunt u een nieuw project starten.
  3. Eerst moet u het programma 'Neopixel' toevoegen aan de programma's die al automatisch in het project staan. Uitleg hierover staat onderaan.
  4. Dan kunt u bovenaan in het midden op 'javascript' klikken.
  5. De code die hier automatisch al staat, moet u eerst weghalen.
  6. Op deze website genereert u code die u op de plaats van de weggehaalde code plakt [Ctrl + V].
  7. Als u de code aan uw project op de website van micro:bit heeft toegevoegd, kunt u de zwarte USB-kabel aansluiten op uw computer of laptop.
  8. De website zal u begeleiden met het verbinden van de computer of laptop met de micro:bit. (De code kan alleen op de microbit gedownload worden als u de browser edge of chrome gebruikt).
  9. Dan kunt u de code op de micro:bit in het powerbankhoesje downloaden.
  10. Als dit gelukt is, kunt u de USB-kabel weer in de powerbank stoppen.
  11. Als u eerst op het knopje B en daarna op het knopje A drukt, zal uw patroon verschijnen.
  12. Wanneer de lampjes aan zijn gegaan, kunt u de zwarte USB-kabel uit de powerbank halen en uw eigen telefoon laden terwijl het hoesje licht geeft!

Hoe u Neopixel kunt toevoegen:

Als u op de website van micro:bit een nieuw project opent, ziet uw scherm er als volgt uit (afbeelding hieronder).

  1. Klik op 'uitbreidingen' (hieronder aangegeven met de kleur geel).
  2. Zoek vervolgens naar 'neopixel'.
  3. Er zal een pakket van neopixel te zien zijn (de volgende afbeelding hieronder).
  4. Als u hierop klikt, zal dit pakket automatisch toegevoegd worden aan uw account.
  5. Nu kunt u uw code toevoegen bij het onderdeel 'javascript'.
