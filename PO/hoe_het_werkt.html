Hoe werkt deze site?

Op deze pagina laten wij u stap voor stap zien hoe u een prachtig lichtgevend powerbankhoesje kunt maken. Hieronder kunt u een patroon maken voor het hoesje. Als u op 'klaar' drukt, zal er een code verschijnen. Deze code moet u goed bewaren.

Hoe werkt de site van micro:bit?

Op deze website gaat het project verder: micro:bit. Een verder stappenplan voor deze website:

  1. U moet eerst een account aanmaken.
  2. Dan kunt u een nieuw project starten.
  3. Dan kunt u bovenaan in het midden op 'javascript' klikken.
  4. Hier kunt u de code invoeren.

Deze website kunt u echter alleen gebruiken met edge of google chrome. Vanwege de verbinding met de microbit die in het hoesje zit.

Hoe nu verder?

Deze code kunt u downloaden naar uw microbit. Hiervoor moet u eerst de microbit in het hoesje aansluiten aan uw computer met de zwarte usb-kabel. Vervolgens kunt u op downloaden klikken. Als het goed is krijgt u nu instructies hoe u de microbit kunt verbinden met de website.

Hoe krijg ik de LED-lampjes nu aan?

Als u het hoesje opent ziet u twee knopjes, knopje A en B. ALs u op knopoje A drukt zullen de lampjes als alle stappen juist zijn gegaan, aangaan. Als dit niet gebeurt kunt u op knopje B drukken en vervolgens opnieuw op knopje A. Nu heeft u een prachtig lichtgevend hoesje.
