Waarom ben ik aan dit project begonnen?

Met het vak Informatica sluit ik mijn eindexamenjaar af met een eindproject. Hierbij mocht ik zelf een project verzinnen. Mij leek het altijd al leuk om iets fysieks te maken. Eerst dacht ik aan het maken van een telefoonhoesje met LED-lampjes. Later werd dit dus een powerbankhoesje. Hierbij kon ik namelijk de batterij van de powerbank gebruiken.

Wat heb ik allemaal geleerd bij dit project?

Ik heb veel verschillende dingen geleerd. Ik begon het project met het maken van een 3D-model. Hierbij heb ik eerst onderzoek gedaan naar hoe ik dit het beste kon doen. Daarna heb ik geleerd hoe ik een basis van een 3D-model kon maken. En uiteindelijk was deze klaar en kon deze uitgeprint worden!
Dit is het eindresultaat!


Daarna ben ik begonnen met het uitzoeken van een chip. Mij werd een microbit aanbevolen. Hier heb ik mij toen verder in verdiept en het bleek een goed toegankelijke chip te zijn. Hier heb ik toen wat dingen mee getest met LED-strips en met de code. Dit ging erg gemakkelijk. Uiteindelijk heb ik grotere stappen gezet en heb ik op een dag alle LED-strips aan elkaar gesoldeerd en heb ik de microbit ingesteld.
