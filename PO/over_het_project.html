Waarom ben ik aan dit project begonnen?

Met het vak Informatica sluit ik mijn eindexamenjaar af met een eindproject. Hierbij mocht ik zelf een project bedenken. Mij leek het altijd al leuk om iets fysieks te maken. Eerst dacht ik aan het maken van een telefoonhoesje met LED-lampjes. Later werd dit een powerbankhoesje. Hierbij kan ik namelijk de batterij van de powerbank gebruiken.

Wat heb ik allemaal geleerd bij dit project?

Ik heb veel verschillende dingen geleerd. Ik begon het project met het maken van een 3D-model. Hierbij heb ik eerst onderzoek gedaan naar welk programma ik hier het beste voor kon gebruiken. Daarna heb ik uitgezocht hoe dit programma werkt en hoe ik dus daadwerkelijk een 3D-model kon maken. En uiteindelijk was dit model klaar en kon deze uitgeprint worden!
Dit is het eindresultaat!


Daarna ben ik begonnen met het uitzoeken van een chip. Mij werd een micro:bit aanbevolen. Hier heb ik mij toen verder in verdiepd en het bleek een toegankelijke chip te zijn. Met deze chip en wat LED-lampjes heb ik wat dingen getest. Hierbij heb ik door verschillende dingen uit te proberen geleerd hoe de code werkt van een micro:bit. Uiteindelijk heb ik grotere stappen gezet en heb ik op een dag alle LED-strips aan elkaar gesoldeerd. Later ben ik nog veel bezig geweest met deze website en de website waarop u zelf een design kunt maken en de code voor u geschreven wordt. En hier is nu het eindresultaat!
